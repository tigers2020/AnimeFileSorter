# 애니메이션 파일 정리 애플리케이션 - 3년차 2분기 개발 계획

## 개발 목표
3년차 2분기는 기본 추천 시스템 개발과 사용자 인터페이스 개선 및 테마 지원 확장에 집중합니다.

- [ ] 사용자 시청 기록 기반 추천 알고리즘 구현
- [ ] 장르/태그 유사성 기반 추천 구현
- [ ] 작품 인기도 및 평점 기반 추천

- [ ] 사용자 피드백 기반 추천 개선 메커니즘
